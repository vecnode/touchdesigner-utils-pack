# touchdesigner-ig20-iegeekcam

IG-20 ieGeek IP camera video stream player for TouchDesigner, tested: 2022.32120. Comes with an `arp` scanner for local network and blob track. The equipment has sensor-triggered night vision shown in the floating window.

```
(change incoming machine at:)
op('/project1/ig20_iegeekcam/video_display/stream_in').par.url
(camera working stream:)
rtsp://192.168.1.x/2
rtsp://192.168.1.x/11
```
